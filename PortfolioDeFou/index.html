Partir de zéro

Passionné d’informatique, mon parcours est avant tout celui d'un défi personnel. En intégrant ma double licence, je partais de zéro : je n’avais jamais écrit une ligne de code, ni étudié l’économie. 02 / 06
Apprentissage fédéré - Prédiction des maladies cardiaques
Projet académique · Graphes et Open Data · Algorithme FedProx

Ce projet utilise l'apprentissage fédéré (via Flower et FedAvg) pour détecter les maladies cardiaques sur le dataset UCI sans centraliser les données, garantissant ainsi la conformité RGPD. Malgré des données hétérogènes (non-IID) entre cinq hôpitaux, L'écart avec le modèle centralisé est de 3,50 %, ce qui valide l'hypothèse. L'architecture, modélisée en étoile avec NetworkX, prouve que la collaboration décentralisée préserve la confidentialité sans sacrifier la performance médicale.

Python PyTorch Flower NetworkX scikit-learn
